# test

Навчальний вебпроєкт, розміщений у публічному GitHub-репозиторії для відпрацювання навичок роботи з системою контролю версій Git та базової організації структури frontend-застосунку.

## Опис проєкту
Проєкт складається з базових файлів клієнтської веброзробки:
- `index.html` — основна HTML-сторінка;
- `style.css` — файл стилів для оформлення інтерфейсу;
- `script.js` — JavaScript-логіка сторінки.

## Мета
Метою проєкту є демонстрація структури простого вебзастосунку та практичне використання GitHub для зберігання, супроводу та впорядкування коду.

## Структура репозиторію
- `index.html` — структура вебсторінки;
- `style.css` — стилізація елементів інтерфейсу;
- `script.js` — програмна логіка сторінки;
- `README.md` — опис проєкту;
- `.gitignore` — перелік файлів, які не повинні потрапляти до репозиторію;
- `LICENSE` — ліцензія на використання коду.
